# Inventory Management Dashboard

## 📌 Project Overview
The Inventory Management Dashboard is an interactive Power BI project designed to monitor inventory performance, stock levels, warehouse operations, and product movement. It helps businesses optimize inventory, reduce stock shortages, and improve operational efficiency through data-driven insights.

## 🎯 Objectives
- Monitor inventory levels.
- Track stock movement.
- Analyze warehouse performance.
- Identify fast-moving and slow-moving products.
- Optimize inventory planning.
- Improve business decision-making.

## 📊 Dashboard Features
- Inventory Overview
- Stock Availability
- Product Category Analysis
- Warehouse Performance
- Inventory Turnover
- Reorder Level Monitoring
- Supplier Analysis
- Interactive Filters and Slicers

## 🛠️ Tools Used
- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel
